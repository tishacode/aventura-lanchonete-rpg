## Sobre o Jogo

“O Mistério do Cofre da Lanchonete” é um jogo de aventura textual onde o jogador explora diferentes salas, interage com objetos e coleta pistas para resolver o enigma: **descobrir a senha do cofre secreto** da lanchonete.

O jogo foi desenvolvido com foco nos princípios da **Programação Orientada a Objetos (POO)**, utilizando **JavaScript** puro e estrutura modular com classes para representar entidades como salas, objetos e ferramentas.

---

## Comandos Disponíveis

Digite os comandos durante o jogo para interagir:

- `ajuda` — lista os comandos disponíveis.
- `olhar` — descreve a sala atual e os itens.
- `ir <direção>` — move o personagem (ex: `ir norte`).
- `pegar <item>` — pega um item da sala.
- `usar <item>` — usa um item do inventário.
- `inventario` — mostra os itens que você tem.
- `sair` — encerra o jogo.
